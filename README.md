# steffensens.io

Personal portfolio website for Elliot Steffensen — software engineer and sailor.

Live at [steffensens.io](https://steffensens.io)

## Stack

- Plain HTML + CSS (no frameworks, no JavaScript)
- Roboto Mono monospace font
- Collapsible sections using the CSS checkbox hack
- Hosted on GitHub Pages with a custom domain

## Structure

```
index.html   — single-page site with all content
style.css    — styling, animations, and CSS reset
CNAME        — custom domain config
```
